``0x10c-asm`` assembly compiler for Notch's DCPU-16
---------------------------------------------------

Install from Pypi:
==================

  pip install 0x10c-asm

Usage:
======

  0x10c-asm.py help

Inspect the generated code:

  0x10c-asm.py <inputfile>

Write the output in a binary file:

  0x10c-asm.py <inputfile> <outputfile>
